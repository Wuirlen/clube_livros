# Lara_Biblioteca
 Sistema de biblioteca com o laravel 7.
 
# Introdução
Esboço de um simples sistema de biblioteca.

# Ações disponiveis
CRUD de usuários, alunos, categorias e livros.

CRUD de consultas dos usuários, alunos, livros, categorias e livros emprestados.
